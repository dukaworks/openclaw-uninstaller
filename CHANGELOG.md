## 🦞💥 OpenClaw Uninstaller 更新日志

## [1.0.0] - 2025-02-26

### ✨ 新增

- 🎨 可爱的 CLI 界面 - 彩色输出 + 动画效果
- 🛑 自动停止服务 - 查找并终止所有 OpenClaw 进程
- 💾 智能备份 - 可选备份配置到 `~/.openclaw_backup/`
- 🗑️ 彻底清理 - 删除所有文件、目录、npm 包
- 🧹 环境清理 - 自动清理 shell 配置中的环境变量
- 🦞 卸载虾主题 - 有仪式感的告别界面
- 📦 pip 安装支持
- 🚀 一键脚本安装

### 🎯 功能

- 交互式向导，防止误删
- 预览模式 (--dry-run)
- 强制模式 (--force)
- 仅备份模式 (--backup-only)
